# Particle
https://root.cern/install/
## Compilazione del progetto
```
cd code
sh runROOT.sh
root [0] .L main.cpp
root [1] main()
(int) 0
root[2] .L analysis.C
root[3] setStyle()
root[4] analysis()
```
